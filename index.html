Hello, I'm Thanasis.

Hi, my name is Athanasios Papias. I'm an Entry-level Software Engineer, Hardware Engineer, Computer Science and IoT Developer.

I can still remember the first time I saw a teacher in junior high school writing some code and sending the code to a small mobile robot. After 12 years, I still have the same excitement when my code returns the correct values from a database or a Buck Converter controlled by a microcontroller returns the correct Voltage in the output. I have participated in many team projects and won some competitions, I was a member and vice president of IEEE Student Branch of University of Ioannina for 2 years.

A very talkative person (as my friends and colleagues say), I love working in a team project and when it comes to writing huge chunks of code, I am pretty lazy, so I will always find a faster and smarter way to do the job. Fluent in Python, C/C++ and Java, currently creating some websites and learning Javascript & Rust.
